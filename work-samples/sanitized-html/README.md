# Sanitoidut portfolio-työnäytteet

Tämä paketti sisältää kolme portfoliota varten sanitoitua HTML-työnäytettä.

Mukana:
1. Sivustokehityksen backlog ja priorisointimalli
2. Orgaanisen näkyvyyden raportti- ja työjonemalli
3. Feedback Workspace

Sanitoinnissa on:
- poistettu tai korvattu työnantaja-/brändiviittauksia
- poistettu upotetut fontit ja logot
- korvattu asiakasdomainit ja sisäiset tiedostonimet esimerkkisisällöllä
- korvattu palautetyöpöydän aineisto synteettisellä demo-datalla
- vaihdettu näkyvässä copyssa vältettäviä termejä selkeämmiksi
- lisätty näkyvä huomautus siitä, että kyseessä on sanitoitu portfolio-demo

Tarkista vielä selaimessa ennen julkaisua.
