# Piwik PRO Workspace

## Ongelma

Analytiikkaraportit kertovat usein, mitä tapahtui, mutta eivät aina auta päättämään, mitä pitäisi tehdä seuraavaksi.

Käyttäjäpolkujen pudotuskohdat, prioriteetit ja jatkotoimet jäävät helposti erillisiksi havainnoiksi.

## Ratkaisu

Piwik PRO Workspace mallintaa paikallisen työtilan, jossa Piwik PRO -data voidaan muuttaa funnel-ajoiksi, pudotuslaskennaksi, diagnoosikorteiksi ja prioriteettitaulukoksi.

Tarkoitus on tehdä analytiikasta operatiivista: ei vain raportointia, vaan päätösten ja jatkotoimien valmistelua.

## Mitä demo näyttää

- Piwik PRO -pohjaisen analytiikkatyön mallinnusta
- funnel-ajojen tallennusta
- pudotuskohtien laskentaa
- diagnoosikortteja
- prioriteettitaulukkoa
- paikallisen dashboard-näkymän

## Mitä osaamista tämä näyttää

- Piwik PRO -analytiikan ymmärrys
- käyttäjäpolkujen ja konversiopolkujen tulkinta
- analytiikan muuttaminen toimenpiteiksi
- paikallinen FastAPI + SvelteKit -työkalusuunta
- mittaamisen ja operatiivisen kehittämisen yhdistäminen

## Arvo asiakkaalle

Demo näyttää, miten analytiikkadata voidaan kääntää käytännön työn tueksi: missä polku vuotaa, mikä on tärkeintä ja mitä kannattaa tarkistaa seuraavaksi.

## Rajaukset

- Demo käyttää mallidataa tai paikallista dataa.
- Ei sisällä julkisia asiakas- tai analytiikkatietoja.
- Ei korvaa analyytikon tulkintaa.
- Tarkoitettu analytiikan työnkulkujen suunnitteluun.
