Kuvakaappausten tarkoitus on näyttää nopeasti:

1. mikä ongelma demossa ratkaistaan
2. miltä työnkulku näyttää
3. mikä on käyttäjän seuraava askel
4. miten tieto muuttuu päätöksen tai tekemisen tueksi

Kuvakaappauksissa ei saa näkyä asiakasdataa, salaisuuksia, API-avaimia, sisäisiä URL-osoitteita tai tunnistettavia henkilötietoja.
